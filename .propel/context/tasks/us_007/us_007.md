# User Story: US-007 - Patient Availability Selection

**Story ID:** US-007  
**Epic:** EP-001 (Patient Experience)  
**Created:** April 17, 2026  
**Status:** Draft  
**Priority:** High  
**Effort:** 4 story points  

---

## User Story

**As a** patient,  
**I want to** view and select available appointment times for a provider,  
**So that** I can choose a convenient slot for my visit.

---

## Acceptance Criteria

### Functional Requirements
- [ ] Display available time slots in calendar or list format
- [ ] Show appointment duration and type for each slot
- [ ] Highlight next available appointment prominently
- [ ] Allow filtering by date range and time preferences (morning/afternoon)
- [ ] Indicate slot availability in real-time (updates every 5 minutes)
- [ ] Show provider's typical wait time or backlog
- [ ] Support booking multiple related appointments

### Non-Functional Requirements
- [ ] Availability loads within 1 second
- [ ] Updates automatically without page refresh
- [ ] Supports 100+ concurrent availability checks

### Edge Cases
- [ ] All slots booked for selected date range
- [ ] Provider schedule changes during selection
- [ ] Time zone differences for patient
- [ ] Appointment types with different durations
