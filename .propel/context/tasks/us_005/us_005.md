# User Story: US-005 - Clinical Intelligence No-Show Prediction

**Story ID:** US-005  
**Epic:** EP-004 (Clinical Intelligence)  
**Created:** April 17, 2026  
**Status:** Draft  
**Priority:** Medium  
**Effort:** 13 story points  

---

## User Story

**As a** clinical scheduler,  
**I want to** see no-show risk predictions for patient appointments,  
**So that** I can proactively intervene to improve attendance rates.

---

## Acceptance Criteria

### Functional Requirements
- [ ] ML model predicts no-show probability for each appointment
- [ ] Risk score displayed in triage and booking interfaces
- [ ] High-risk patients (>65% probability) flagged automatically
- [ ] Automatic double-reminder scheduling for high-risk appointments
- [ ] Staff can override predictions with clinical judgment
- [ ] Model accuracy tracking and reporting
- [ ] Weekly model retraining with new appointment data

### Non-Functional Requirements
- [ ] Prediction calculation within 500ms
- [ ] Model accuracy target >75% precision
- [ ] Supports 10,000 predictions per minute
- [ ] 99.9% uptime for prediction service

### Edge Cases
- [ ] New patients with no historical data
- [ ] Patients with conflicting appointment patterns
- [ ] Seasonal or event-based attendance variations
- [ ] Model drift requiring retraining
