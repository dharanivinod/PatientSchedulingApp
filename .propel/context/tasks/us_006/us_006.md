# User Story: US-006 - Patient Provider Search

**Story ID:** US-006  
**Epic:** EP-001 (Patient Experience)  
**Created:** April 17, 2026  
**Status:** Draft  
**Priority:** High  
**Effort:** 3 story points  

---

## User Story

**As a** patient,  
**I want to** search for healthcare providers by specialty and location,  
**So that** I can find appropriate doctors for my needs.

---

## Acceptance Criteria

### Functional Requirements
- [ ] Search input accepts specialty keywords (e.g., "cardiology", "pediatrics")
- [ ] Location filter supports city, zip code, or "near me" geolocation
- [ ] Results display provider name, specialty, location, and distance
- [ ] Search results update in real-time as user types
- [ ] Support for advanced filters: gender, language, insurance accepted
- [ ] Empty state shows helpful guidance when no results found
- [ ] Recent searches saved for quick access

### Non-Functional Requirements
- [ ] Search completes within 500ms for up to 1000 providers
- [ ] Supports fuzzy matching for specialty names
- [ ] Accessible with keyboard navigation and screen readers

### Edge Cases
- [ ] No providers available in selected location
- [ ] Specialty not offered at any location
- [ ] Geolocation permission denied
- [ ] Network connectivity issues during search

## Visual Design Context

### Screen References
- SCR-001: Patient Landing / Home (search entry)
- SCR-002: Patient Search & Availability (results display)

### Key UI Elements
- Search bar: Large input with search icon and clear button
- Filter chips: Specialty, location, insurance (removable)
- Provider cards: Compact list with photo, name, specialty, rating
- Loading state: Skeleton cards during search
- Error state: Friendly message with retry option

### Design Tokens
- Search input: Full-width with subtle border (#CBD2DE)
- Primary action: Blue search button (#3B6FB7)
- Result cards: White background with shadow

### Responsive Behavior
- Desktop: Sidebar filters with main results grid
- Mobile: Stacked filters above results list
- Tablet: Collapsible filter panel
