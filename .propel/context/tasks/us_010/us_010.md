# User Story: US-010 - Patient Appointment Rescheduling

**Story ID:** US-010  
**Epic:** EP-001 (Patient Experience)  
**Created:** April 17, 2026  
**Status:** Draft  
**Priority:** High  
**Effort:** 4 story points  

---

## User Story

**As a** patient,  
**I want to** reschedule my existing appointment to a different time,  
**So that** I can attend at a more convenient time.

---

## Acceptance Criteria

### Functional Requirements
- [ ] Allow rescheduling up to 24 hours before appointment
- [ ] Show alternative available slots for same provider
- [ ] Display reason selection for rescheduling
- [ ] Prevent rescheduling within 24-hour window
- [ ] Send confirmation of rescheduled appointment
- [ ] Release original slot for other patients
- [ ] Update all reminder notifications

### Non-Functional Requirements
- [ ] Reschedule completes within 3 steps
- [ ] New appointment confirmed within 1 minute
- [ ] Original slot released immediately

### Edge Cases
- [ ] Attempt to reschedule within 24 hours
- [ ] No alternative slots available
- [ ] Provider no longer available
- [ ] Conflicting reschedule requests
